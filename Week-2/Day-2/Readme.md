# Week 2 - Day 2

Today I solved 3 Sliding Window problems from LeetCode.

## Problems Solved

### 1. Longest Substring Without Repeating Characters
- Used Sliding Window and Hash Map.
- Tracked the last position of each character to avoid duplicates.

Time Complexity: O(n)
Space Complexity: O(n)

### 2. Permutation in String
- Used Sliding Window with frequency arrays.
- Compared character frequencies of the current window with the target string.

Time Complexity: O(n)
Space Complexity: O(1)

### 3. Find All Anagrams in a String
- Used Sliding Window and frequency arrays.
- Stored starting indices of all matching anagrams.

Time Complexity: O(n)
Space Complexity: O(1)

## Concepts Learned
- Sliding Window
- Hash Map
- Frequency Array
- String Processing
- Window Expansion and Shrinking
