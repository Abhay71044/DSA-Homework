# Week 1 - Day 1

Today I solved 3 array problems from LeetCode.

## Problems Solved

### 1. Two Sum
- Used brute force approach with nested loops.
- Checked every pair of elements to find the target sum.

Time Complexity: O(n²)
Space Complexity: O(1)

### 2. Remove Duplicates from Sorted Array
- Used Two Pointer technique.
- Stored unique elements at the start of the array.

Time Complexity: O(n)
Space Complexity: O(1)

### 3. Best Time to Buy and Sell Stock
- Kept track of the minimum stock price.
- Calculated maximum profit while traversing the array.

Time Complexity: O(n)
Space Complexity: O(1)

## What I Learned
- Basics of Arrays
- Two Pointer Technique
- Greedy Approach
- How to optimize solutions
- Time and Space Complexity analysis
