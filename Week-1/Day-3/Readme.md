# Week 1 - Day 3

Today I solved 3 problems using Two Pointer technique.

## Problems Solved

### 1. Move Zeroes
- Moved all zeroes to the end of the array.
- Maintained the relative order of non-zero elements.

Time Complexity: O(n)
Space Complexity: O(1)

### 2. Squares of a Sorted Array
- Used Two Pointer approach.
- Compared squares from both ends and filled the answer array from the back.

Time Complexity: O(n)
Space Complexity: O(n)

### 3. Container With Most Water
- Used Two Pointers starting from both ends.
- Calculated area and moved the pointer with smaller height.

Time Complexity: O(n)
Space Complexity: O(1)

## Concepts Learned
- Two Pointers
- Array Traversal
- Optimization Techniques
- Time and Space Complexity Analysis
