# Week 1 - Day 5

Today I solved 3 string-based problems from LeetCode.

## Problems Solved

### 1. Valid Palindrome
- Used Two Pointer approach.
- Ignored special characters and compared characters in a case-insensitive manner.

Time Complexity: O(n)
Space Complexity: O(1)

### 2. Reverse String
- Used Two Pointers to swap characters from both ends of the string.
- Reversed the string in-place.

Time Complexity: O(n)
Space Complexity: O(1)

### 3. Longest Common Prefix
- Compared characters of all strings index by index.
- Stopped when a mismatch was found.

Time Complexity: O(n × m)
Space Complexity: O(1)

## Concepts Learned
- Strings
- Two Pointer Technique
- Character Manipulation
- Case-Insensitive Comparison
- Common Prefix Matching
